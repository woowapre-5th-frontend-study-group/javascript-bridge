# 구현기능목록

## 다리길이 입력 기능

- 3이상 20이하의 숫자만 입력받을 것.
- 예외처리
  > 3 ~ 20사이의 숫자가 아닐 경우
  > 숫자가 아닌경우

## 이동할 칸 입력 기능

- 위일 경우 U, 아래일 경우 D를 입력받을 것
- 예외처리
  > U 혹은 D가 아닐 경우

## 맞추지 못할 경우 재시도 여부 입력 기능

- 예외처리
  > R 혹은 Q가 아닐 경우
- 종료일 경우 최종 게임 결과 출력
